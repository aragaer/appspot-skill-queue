What is this?

This is an online tool for tracking skill queues of your characters.

What does it do?

It tracks the skill queue of your characters and sends you an e-mail notification once their skill queues are about to expire.

How can I start using it?

You need a google account to login (I'll add OpenID later) and you'll have to create an API key giving access to your characters' skill queues. Don't worry, it's easier than it sounds.

How does it work?

API, cron, Python. The source code is here.

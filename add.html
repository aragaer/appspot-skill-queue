What's that now?

You might need to login first. Fear not, that is indeed Eve Online site. You can still open it in other window if you have any doubts.

That's where you create an API key for use with the Skill Queue Tracker. The predefined access mask is already set. All you need is to enter a name for the key, then submit it (scroll to bottom).

If you already have a key that will work, you can use the 'Back to list' link at the bottom of the page.

Once at the key list, find the key you have created, then copy and paste ID and verification code to the fields in the top left corner.
